# Student Planner Web Application (PHP & MySQL)
PHP MySQL HTML CSS JavaScript AJAX

This project is a full-stack web-based Student Planner system designed to help university students manage academic and personal tasks efficiently. The application provides task tracking, prioritization, reminders, and secure user management through a clean and user-friendly interface.

---

## Overview

The Student Planner Web Application centralizes task management for students by allowing them to record, monitor, prioritize, and archive tasks. The system is built using PHP and MySQL for backend processing, with HTML, CSS, JavaScript, and AJAX handling the frontend interaction.

The application runs on a local development environment (localhost) and is suitable for academic projects, demonstrations, and beginner-to-intermediate web development learning.

---

## Key Features

- Secure user authentication (Login, Register, Forgot Password)
- Task recording with title, description, due date, category, priority, and status
- Task monitoring and sorting by due date and priority
- Task status management (On-going, Pending, Completed)
- Automatic task archiving for completed and deleted tasks
- Reminder system with popup notifications and notification page
- User profile management (edit profile and delete account)
- Session-based access control
- Centralized database configuration
- Dark-themed modern user interface

---

## System Workflow

1. User creates the database using the provided MySQL file.
2. User connects the project to localhost.
3. User registers and logs in to the system.
4. Tasks are added, edited, viewed, and tracked through the dashboard.
5. Completed tasks are archived automatically.
6. Notifications alert users of upcoming deadlines.

## Security Notes

- Passwords are encrypted using `password_hash()` and `password_verify()`
- Prepared statements prevent SQL injection
- Session-based authentication protects restricted pages
